# Concrete-Column-Heat-Maps
# Concrete Column Damage Predictor

This project uses machine learning models to predict the damage level of concrete columns based on heatmaps.

## Model Files

The trained models are hosted on Google Drive and can be downloaded using the links below:

### Models Based on Drift:
- [Model 1](https://drive.google.com/uc?id=1Kl9AnoERY4twkWyM9W0f-tbsE5nxUrSa&export=download)
- [Model 2](https://drive.google.com/uc?id=13cG3QPOz0BZAauSMMLyNCsZg55oqnE6i&export=download)
- [Meta Model](https://drive.google.com/uc?id=1hVYK3o0I1AK8WWT__4urO8isLpis1Udx&export=download)

### Models Based on DIS:
- [Model 1](https://drive.google.com/uc?id=12w2fGQGbuzKxdKBtSws5NIA2I_AXAWoV&export=download)

## Usage

1. Download the models from the links above.
2. Place the downloaded `.h5` files in the `models/` directory.
3. Run the Streamlit app:
   ```bash
   streamlit run app.py
